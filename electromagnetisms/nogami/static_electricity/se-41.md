# 静電気
## 1.41 鏡像法の乱用

平面について鏡像法を使う。そうすると、$x,y$軸の$-$部分で電位がゼロにならないのでまたあたらしく第3象限に電荷をおいてやる。4つの電荷のつくる電位を計算すればよいことになる。
