# 熱力学の基礎
## 1.1 電場がゼロになる点

電荷の大小から、小さい方の電荷の左側に電場がゼロの点ができるのは明確。
<br>
よってその点を$x_1$としてそこからの距離で電場を計算すれば$ (x_1-0.2) ^2 $とかいう面倒な項は出てこない。どこから座標をとるかはめちゃくちゃ大事。
