# 静電気
## 1.28 平行版コンデンサーに導体を挿入


最初に金属板を誘電体として計算するとかいう意味のわからないことをして失敗した。導体を挿入するとその厚さだけの電位が減少する。すなわち、静電容量が増加する。
<br>
書いていないが、絶縁導体板上には$ \pm Q $の電荷が誘導される。
