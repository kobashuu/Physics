# 静電気
## 1.36 電場の連続性 

電場の法線成分の連続性から等式を立てる。また、また接線成分について$ \mathrm{rot} \mathbf{E} = 0 $の条件から立式して連立して解く。
<br>
本当は境界面の微小円筒を考えてガウスの法則を使って法線成分の等式を導かないといけない。
