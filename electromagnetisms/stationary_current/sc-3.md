# 定常電流
## 2.3 コンデンサー⇄抵抗

最初に理想導体間を電気伝導率$ \sigma $の媒質で満たした場合について、電極上でガウスの法則を使って全電流を導く。
<br>
その後、電位差Vを一定に保ったまま誘電率$ \varepsilon $の媒質に入れ替えると、オームの法則から静電容量と抵抗の関係式を導くことができる。
